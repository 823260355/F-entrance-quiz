# Setup

## Start Server

Server started at `http://localhost:1234` . Parcel is used for bundler.

```
npm run start
```

## Run Jest Test

Jest is used for Unit Test.

```
npm run test
```

### Run Cypress Test with Web Server started

```
npm run cypress
```

### Open Cypress CLI

```
npm run cypress:open
```

### Run Cypress Test

```
npm run cypress:run
```
